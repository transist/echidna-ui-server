# Echidna UI

Testing server for  Echidna UI components
Server is hosted in the branch server

To use it :

    git clone git@github.com:transist/echidna-ui.git
    
    // checkout the server
    cd echidna-ui
    git checkout server

    // add the public folder
    git clone git@github.com:transist/echidna-ui.git

    // launch the app
    node app.js
